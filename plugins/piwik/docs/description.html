Allows to insert Piwik Analytics code.

TO INSERT A WIDGET INTO YOUR SITE:

  • Go to «Personal Settings» page (user icon on the right top menu);
  • Click on «Widgets» in left sidebar menu;
  • Choose a widget you would like to add;
  • Copy the Embed Iframe code;
  • Go to «Manage Blocks» page in Subrion Admin Panel;
  • Create a new block with 'html' type, click 'Source' button in the 'Contents' area, and paste the code there;
  • Save the changes and check your front-end
